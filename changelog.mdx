Stay up to date with what's new in SaveMate. Major releases are also announced on the [SaveMate Blog](https://savemate.io/blog).

## February 2026

### Cloud Storage Connection Health

- **OAuth token resilience** — SaveMate now proactively refreshes OAuth tokens for Google Drive and Dropbox before they expire
- **Connection health indicators** — Cloud connections show their status as healthy, broken, or revoked so you can take action before uploads fail
- **Automatic recovery** — Temporary auth failures are retried automatically without disconnecting your account

### Public Downloads Page

- **New public downloads page** — Access SaveMate downloads directly at [savemate.io/downloads](https://savemate.io/downloads) without navigating through the dashboard

### Improved Error Handling

- **Rate limit feedback** — Clear toast notifications when you hit download or API rate limits, with guidance on when to retry
- **Standardized notifications** — Consistent toast notification style across all features with title and description

### Review Modal

- **In-extension feedback** — After reaching download milestones, SaveMate gently asks for a Chrome Web Store review to help other users discover the extension

### Internationalization

- **23 languages supported** — The extension popup and options page are now available in 20+ languages including French, Arabic, Spanish, German, Japanese, Korean, and more
- **Chrome Web Store translations** — Store listing translated for 18 locales

## January 2026

### Universal Stream Detection v2.0

- **Enhanced Vimeo detection** — Dedicated detector intercepts Vimeo's player config API for accurate quality and file size information
- **DASH manifest parsing** — Full support for DASH (MPD) streams alongside HLS
- **HLS bandwidth sampling** — Intelligent quality detection that samples stream bandwidth for accurate file size estimates
- **SPA navigation support** — Detection works correctly on single-page applications (React, Vue, Angular) that don't trigger full page reloads

### Post-Install Welcome Page

- **Guided onboarding** — New users see a welcome page after installation with a quick overview of features and next steps

### Extension Settings Improvements

- **Parallel settings loading** — Settings load faster with concurrent initialization
- **Download path customization** — Set a custom folder prefix to keep SaveMate downloads organized

## December 2025

### Cloud Storage Sync

- **Google Drive** — OAuth-based upload with resumable upload support
- **Dropbox** — OAuth-based upload with standard API
- **AWS S3** — Credential-based multipart upload for large files
- **Cloudflare R2** — S3-compatible upload with cost-effective pricing
- **Pause and resume** — Multipart uploads can be paused and resumed at any time
- **Upload history** — Track all uploads with status filtering and retry capability

### Chromecast & AirPlay

- **Cast to TV** — Stream detected videos to Chromecast and AirPlay devices on your local network
- **Cast player** — Full playback controls with quality selection and session management

### Format Conversion

- **Container conversion** — Convert HLS and DASH streams to MP4, MKV, MOV, or WebM
- **Audio extraction** — Extract audio tracks to MP3 or AAC
- **Local processing** — All conversion happens in your browser using WebAssembly — no server uploads

## November 2025

### Video Library

- **Cloud library** — Save detected videos to your account for access from any device
- **Cross-device sync** — Access your saved videos on desktop, laptop, or mobile web
- **Search and sort** — Find videos by title, sort by date or name

### Billing & Subscription

- **Professional plan** — Monthly, yearly, and lifetime billing options
- **Lifetime access** — One-time payment for permanent Professional access
- **Stripe integration** — Secure payment processing via Stripe

## October 2025

### Initial Release

- **SaveMate v1.0** — Universal video stream detector for Chrome, Edge, and Brave
- **HLS detection** — Automatic detection of m3u8 streams with quality selection
- **Progressive downloads** — Direct download of MP4, WebM, and 16+ video formats
- **Free plan** — Unlimited 720p downloads, 3 HD downloads per day
